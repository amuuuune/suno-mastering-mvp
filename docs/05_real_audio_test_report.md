## 判定

合格です。

- 5曲すべてでWAVのIntegrated LUFSが目標 -14 LUFS付近に収まりました。
- 5曲すべてでWAVのTrue Peakが -1 dBTP以下でした。
- 5曲すべてでMP3コピーも -14 LUFS付近に収まりました。
- MP3コピーでもTrue Peakは -1 dBTP以下でした。
- 日本語ファイル名の音源も、バックエンドAPI検査では問題なく処理できました。

## 残る確認

この検査で分かるのは、数値、出力成功、ファイルサイズ、API経路の健全性です。

次に必要なのは耳での確認です。特に、プリセットごとの効き方、低域の残し方、ボーカルの痛さ、Suno特有の高域のザラつきが好みに合うかは、実際に再生して判断します。
